Promise

  1. Promise is native to javascript
  2. Promise is eager it returns the data as soon as promise is created
  3. Promise can emit only single values
  4. Promise has method for succes and error
  5. Promise always returns asynchrounous data

Observable

  1. Observable is not native to js and imported from rxjs library
  2. observable is lazy its only emits data when we subsribe to observable
  3. Observable can emit stream of values or multiple values
  4. Observable has methods for success, error and completion
  5. Observable returns both synchronous and async data
